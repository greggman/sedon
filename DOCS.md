# Sedon Docs

## Terse instructions:

* <kbd>Ctrl<kbd>-</kbd>Shift</kbd>-<kbd>P</kbd> / <kbd>Cmd</kbd>-</kbd>Shift<kbd>-<kbd>P</kbd> pulls up a command menu similar to VSCode

* To Edit a SubGraph, click "edit" or double click it

* To Preview a SubGraph, right click on it in the Assets view and pick "Open in Canvas" or choose it from the drop dawn in the Preview window.

* To Add a SubGraph to a graph, drag it from the Asset view

* To display more than one object add a [scene-merge](https://greggman.github.io/sedon/docs/nodes/core/scene-merge/index.html).

* To create a subgraph pick Add->New Subgraph from the menus

* You can <kbd>shift</kbd> drag a selection box to select nodes

* Pick `View->Cleanup` to try to auto-layout a graph (needs work! 😭)

* Nodes have a `?` for help

* <kbd>f</kbd> frames the current selection. Works in both the Canvas (the flowchart), and the preview.

* Camera Controls
  * Drag orbits the camera around its target
  * <kbd>ctrl</kbd> drag or <kbd>cmd<kbd> drag pans the camera
  * Mousewheel dollies the camera toward / away from current target
  * <kbd>w</kbd> / <kbd>s</kbd> move the camera forward, backward at the current elevation
  * <kbd>a</kbd> / <kbd>w</kbd> move the camera left / right
  * <kbd>q</kbd> / <kbd>q</kbd> move the camera up / down
